Python Calculator (Module 5 Assignment)
=======================================

Overview
--------
This project is a fully functional calculator built in Python for the Module 5 assignment.
It begins with simple arithmetic functions and gradually evolves into a complete, object-oriented, menu-driven calculator application.

Features
--------
• Performs addition, subtraction, multiplication, and division.
• Handles division-by-zero safely.
• Validates user input to prevent crashes.
• Uses clear menus for continuous calculations until exit.
• Object-oriented design with a Calculator class for modularity and clarity.

How To Run
----------
1. Make sure Python is installed on your computer.
2. Clone or download this repository:
   git clone https://github.com/utsho261/PythonCalculator.git
3. Open the project folder in your terminal or command prompt.
4. Run the program:
   python calculator.py

Example Usage
-------------
When you run the program, you will see a menu like this:

1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit

Choose an operation, enter two numbers, and the result will be displayed.
The program continues running until you choose the "Exit" option.
